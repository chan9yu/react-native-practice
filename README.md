## Clean Guide

### Android

1. android 디렉토리 이동

```bash
cd android
```

2. Gradle Clean

```bash
./gradlew clean
```

3. 프로젝트 루트로 이동

```bash
cd ..
```

### iOS

1. ios 디렉토리 이동

```bash
cd ios
```

2. CocoaPods 캐시 클린

```bash
pod cache clean --all
```

3. Pod 설치

```bash
pod install
```

4. 프로젝트 루트로 이동

```bash
cd ..
```
